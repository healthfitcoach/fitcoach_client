# UC19 ViewPointHistory - 포인트를 조회한다

## Basic Path

1. 고객은 내정보 화면에서 [포인트 조회] 메뉴를 선택한다.
2. 시스템은 고객 계정의 포인트 정보를 조회한다.
3. 시스템은 현재 포인트 잔액과 유효기간 정보를 화면 상단에 출력한다.
4. 시스템은 포인트 적립/사용 내역 목록(날짜, 구분, 사유, 포인트 변동량, 잔여포인트)을 최신순으로 출력한다.
5. 고객은 포인트 잔액 및 내역을 확인한다. (A1)
6. 고객은 특정 기간의 포인트 내역을 확인하기 위해 기간 필터를 설정한다. (A2)
7. 시스템은 설정된 기간에 해당하는 포인트 내역을 필터링하여 출력한다.
8. 고객은 포인트 내역 중 특정 항목을 선택하여 상세 정보를 확인한다.
9. 시스템은 선택한 포인트 내역의 상세 정보(적립/사용 일시, 관련 거래, 상세 사유)를 출력한다.
10. 시스템은 포인트 소멸 예정 내역이 있는 경우 "N일 후 N포인트가 소멸 예정입니다" 안내 메시지를 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 포인트 잔액이 0인 경우
  1. 시스템은 "현재 보유한 포인트가 없습니다" 메시지와 포인트 적립 방법 안내를 출력한다.
  2. 고객은 안내 내용을 확인한다.
  3. 유스케이스를 종료한다.

A2. 기간 필터 설정
  1. 고객은 조회 기간(1개월, 3개월, 6개월, 직접 입력)을 선택한다.
  2. 시스템은 선택한 기간의 포인트 내역을 조회하여 출력한다.
  3. 5단계로 돌아간다.

## Exception

E1. 포인트 정보 로딩 실패
  1. 시스템은 "포인트 정보를 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 포인트 내역 조회 실패
  1. 시스템은 "포인트 내역을 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
