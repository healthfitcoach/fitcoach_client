# UC18 ManageAdditionalProduct - 부가적인 상품을 관리한다

## Basic Path

1. 고객은 회원권 관리 화면에서 [부가 상품 관리] 버튼을 선택한다.
2. 시스템은 현재 고객이 이용 중인 부가 상품 목록(락커 번호, 운동복 대여 여부, PT 이용권 수)을 출력한다.
3. 시스템은 추가로 신청 가능한 부가 상품 목록과 가격을 함께 출력한다.
4. 고객은 신규 부가 상품을 선택하고 [신청하기] 버튼을 누른다. (A1)
5. 시스템은 선택한 부가 상품의 상세 정보(이용 기간, 가격, 조건)와 함께 신청 확인 화면을 출력한다.
6. 고객은 신청 내용을 확인하고 [결제 및 신청] 버튼을 누른다.
7. 시스템은 결제 유스케이스(UC07 Pay)를 실행한다.
8. 시스템은 결제 완료 후 부가 상품을 고객 계정에 등록한다.
9. 시스템은 "부가 상품 신청이 완료되었습니다" 메시지와 신청 내역을 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 이미 동일한 부가 상품 이용 중
  1. 시스템은 "이미 해당 부가 상품을 이용 중입니다. 추가 구매하시겠습니까?" 메시지를 출력한다.
  2. 고객은 추가 구매 여부를 결정한다.
  3. 추가 구매 허용 시 4단계로 돌아간다.
  4. 추가 구매 거부 시 화면을 닫고 2단계로 돌아간다.

## Exception

E1. 부가 상품 정보 로딩 실패
  1. 시스템은 "부가 상품 정보를 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 결제 완료 후 부가 상품 등록 실패
  1. 시스템은 "부가 상품 등록에 실패하였습니다. 고객센터에 문의해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E3. 활성 회원권 없이 부가 상품 신청 시도
  1. 시스템은 "부가 상품은 활성화된 회원권 보유 시에만 신청 가능합니다" 메시지를 출력한다.
  2. 유스케이스를 종료한다.
