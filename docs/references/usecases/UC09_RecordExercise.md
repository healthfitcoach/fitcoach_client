# UC09 RecordExercise - 운동기록을 기록한다

## Basic Path

1. 고객은 앱 메인 화면에서 [운동기록] 메뉴를 선택한다.
2. 시스템은 운동 기록 입력 화면(날짜, 운동 종류, 운동 시간, 세트 및 반복 수, 메모)을 출력한다.
3. 고객은 운동 날짜를 선택한다. (A1)
4. 고객은 수행한 운동 종류를 선택하거나 직접 입력한다.
5. 고객은 운동 시간, 세트 수, 반복 횟수 등 운동 세부 정보를 입력한다. (A2)
6. 시스템은 입력된 정보의 유효성(필수 항목 입력 여부, 수치 범위)을 검증한다.
7. 고객은 추가 메모 또는 운동 사진을 선택적으로 첨부한다. (A3)
8. 고객은 [기록 저장] 버튼을 누른다.
9. 시스템은 운동 기록을 저장하고 포인트 지급 유스케이스(UC10 EarnPoints)를 실행한다.
10. 시스템은 저장 완료 메시지와 함께 지급된 포인트 내역을 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 오늘 날짜 자동 선택
  1. 시스템은 오늘 날짜를 기본값으로 자동 설정한다.
  2. 4단계로 돌아간다.

A2. 운동 종류 복수 추가
  1. 고객은 [운동 추가] 버튼을 눌러 추가 운동 항목을 입력한다.
  2. 시스템은 추가 운동 입력 폼을 출력한다.
  3. 고객은 추가 운동 정보를 입력한다.
  4. 6단계로 돌아간다.

A3. 추가 메모, 운동 사진 첨부
  1. 고객은 추가 메모 혹은 운동 사진을 첨부한다.
  2. 시스템은 입력받은 데이터를 해당 날짜 운동기록에 저장한다.
  3. 7단계로 돌아간다.

## Exception

E1. 필수 항목 미입력
  1. 시스템은 "운동 종류와 운동 시간은 필수 입력 항목입니다" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 운동 기록 저장 실패
  1. 시스템은 "운동 기록 저장에 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E3. 회원권 미보유 상태에서 기록 시도
  1. 시스템은 "활성화된 회원권이 없습니다. 회원권을 먼저 구매해주세요" 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E4. 입력 데이터 유효성 검증 실패
  1. 시스템은 "유효하지 않은 입력값입니다. 운동 횟수는 숫자로 작성하여 주세요." 메시지를 출력한다.
  2. 유스케이스를 종료한다.
