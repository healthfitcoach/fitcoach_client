# UC03 PurchaseMembership - 회원권을 구매한다

## Basic Path

1. 고객은 구매 카테고리 목록에서 [회원권] 항목을 선택한다.
2. 시스템은 이용 가능한 회원권 목록(1개월권, 3개월권, 6개월권, 1년권)과 각 가격 및 혜택을 출력한다.
3. 고객은 원하는 회원권 상품을 선택한다.
4. 시스템은 선택한 회원권의 상세 정보(이용 기간, 이용 가능 시설, 가격)를 출력한다.
5. 고객은 상세 정보를 확인하고 [구매하기] 버튼을 누른다.
6. 시스템은 현재 활성화된 회원권 보유 여부를 확인한다. (A1)
7. 고객은 회원권 시작일을 선택한다. (A2)
8. 시스템은 선택한 회원권 정보와 최종 결제 금액을 포함한 주문 확인 화면을 출력한다.
9. 고객은 주문 내역을 확인하고 결제 진행에 동의한다.
10. 시스템은 결제 유스케이스(UC07 Pay)를 실행하고, 결제 완료 후 회원권을 계정에 등록하며 "회원권 구매가 완료되었습니다" 메시지를 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 이미 활성화된 회원권이 존재하는 경우
  1. 시스템은 활성화된 회원권의 정보를 출력한다.
  2. 고객은 현재 보유 중인 회원권의 정보를 확인한다.
  3. 시스템은 "기존 회원권 만료 후 적용됩니다." 안내 메시지를 출력한다.
  4. 고객은 연장 구매 여부를 승인한다.
  5. 7단계로 돌아간다.

A2. 즉시 시작 선택
  1. 시스템은 오늘 날짜를 시작일로 자동 설정한다.
  2. 8단계로 돌아간다.

## Exception

E1. 회원권 등록 실패 (결제 완료 후 시스템 오류)
  1. 시스템은 "회원권 등록에 실패하였습니다. 고객센터에 문의해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 판매 종료된 회원권 선택
  1. 시스템은 "해당 회원권은 현재 판매되지 않습니다" 메시지를 출력한다.
  2. 유스케이스를 종료한다.
