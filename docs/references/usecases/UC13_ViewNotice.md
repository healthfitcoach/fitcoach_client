# UC13 ViewNotice - 공지사항을 확인한다

## Basic Path

1. 고객은 앱 메인 화면에서 [공지사항] 메뉴를 선택한다.
2. 시스템은 공지사항 목록(제목, 작성일, 카테고리, 읽음 여부)을 최신순으로 출력한다.
3. 고객은 목록을 스크롤하며 확인한다.
4. 고객은 확인하고자 하는 공지사항을 선택한다.
5. 시스템은 선택한 공지사항의 상세 내용(제목, 작성일, 내용, 첨부파일)을 출력한다.
6. 시스템은 해당 공지사항을 읽음 처리하고 읽음 상태를 업데이트한다.
7. 고객은 공지사항 내용을 확인한다. (A1)
8. 고객은 [이전 공지] 또는 [다음 공지] 버튼을 통해 인접한 공지사항으로 이동한다. (A2)
9. 고객은 공지사항 확인을 완료하고 [목록으로] 버튼을 누른다.
10. 시스템은 공지사항 목록 화면으로 돌아가며 유스케이스를 종료한다.

## Alternative Path

A1. 첨부파일 다운로드 선택
  1. 고객은 공지사항 내 첨부파일을 선택한다.
  2. 시스템은 첨부파일을 다운로드하여 기기에 저장한다.
  3. 7단계로 돌아간다.

A2. 인접 공지가 없는 경우
  1. 시스템은 "이전(또는 다음) 공지사항이 없습니다" 메시지를 출력한다.
  2. 7단계로 돌아간다.

## Exception

E1. 공지사항 목록 로딩 실패
  1. 시스템은 "공지사항을 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 공지사항 상세 내용 로딩 실패
  1. 시스템은 "해당 공지사항을 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
