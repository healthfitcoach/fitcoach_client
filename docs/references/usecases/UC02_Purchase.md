# UC02 Purchase - 구매한다

## Basic Path

1. 고객은 앱 메인 화면에서 [구매] 메뉴를 선택한다.
2. 시스템은 구매 가능한 카테고리 목록(회원권, 헬스 프로그램, 운동용품, PT, 이벤트 중인 상품)을 출력한다.
3. 고객은 구매하고자 하는 카테고리를 선택한다. (A1)
4. 시스템은 선택한 카테고리에 해당하는 상품 목록과 상세 정보(가격, 설명)를 출력한다.
5. 고객은 목록에서 원하는 상품을 선택한다.
6. 시스템은 선택한 상품의 상세 페이지(이름, 가격, 조건, 혜택)를 출력한다.
7. 고객은 상품 내용을 확인하고 [구매하기] 버튼을 누른다.
8. 시스템은 해당 카테고리에 맞는 구매 유스케이스(UC03/UC04/UC05/UC06)로 흐름을 이동한다.
9. 고객은 결제 절차를 진행한다.
10. 시스템은 결제 결과를 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 구매 가능한 상품이 없는 카테고리 선택
  1. 시스템은 "현재 해당 카테고리에 판매 중인 상품이 없습니다" 메시지를 출력한다.
  2. 고객은 다른 카테고리를 선택한다.
  3. 3단계로 돌아간다.

## Exception

E1. 비로그인 상태에서 구매 시도
  1. 시스템은 "로그인이 필요한 서비스입니다" 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 상품 정보 로딩 실패
  1. 시스템은 "상품 정보를 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
