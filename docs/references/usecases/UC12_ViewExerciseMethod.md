# UC12 ViewExerciseMethod - 운동방법을 조회한다

## Basic Path

1. 시스템은 기구검색 유스케이스(UC11)로부터 선택된 기구 및 운동 정보를 전달받아 진입 조건을 확인한다.
2. 시스템은 선택한 운동의 상세 정보(운동 이름, 대상 부위, 난이도, 준비 자세, 단계별 운동방법)를 출력한다.
3. 고객은 선택한 운동의 상세 정보를 확인한다.
4. 시스템은 운동방법 설명과 함께 참고 이미지 또는 영상을 출력하며 유스케이스를 종료한다.

## Exception

E1. 운동방법 콘텐츠 로딩 실패
  1. 시스템은 "운동방법 정보를 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 영상 재생 실패
  1. 시스템은 "동영상을 불러오는 데 실패하였습니다. 네트워크 연결을 확인해주세요" 오류 메시지를 출력한다.
  2. 시스템은 이미지 및 텍스트 설명만 출력하며 계속 진행한다.
