# UC20 SchedulePT - PT 일정을 잡는다

## Basic Path

1. 고객은 앱 메인 화면에서 [PT 일정 예약] 메뉴를 선택한다.
2. 시스템은 고객의 이용권에 등록된 트레이너의 스케줄을 출력한다. (A1, E1)
   - 예약 가능한 슬롯: 선택 가능 상태로 출력
   - 이미 예약된 슬롯: 선택 불가 상태로 출력
3. 고객은 예약 가능한 슬롯 중 PT 희망 일정(날짜 및 시간)을 선택한다.
4. 시스템은 선택한 트레이너명, 날짜, 시간을 포함한 예약 확인 화면을 출력한다.
5. 고객은 예약 내역을 확인하고 [예약 확정] 버튼을 선택한다.
6. 시스템은 예약 정보를 저장하고 트레이너에게 신규 예약 알림을 전송한다. (E2)
7. 시스템은 트레이너가 수락 시 "PT 일정이 예약되었습니다" 메시지를 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 등록된 PT 이용권 없음
  1. 시스템은 "등록된 PT 이용권이 없습니다. PT를 먼저 구매해주세요" 메시지를 출력한다.
  2. 유스케이스를 종료한다.

## Exception

E1. 트레이너 스케줄 로딩 실패
  1. 시스템은 "트레이너 스케줄 정보를 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 예약 저장 실패
  1. 시스템은 "예약 처리 중 오류가 발생하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
