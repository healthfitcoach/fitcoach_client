UC01 SignUp - 회원가입을 한다

---

Basic Path

1. 고객은 앱 메인 화면에서 [회원가입] 버튼을 선택한다.
2. 시스템은 회원가입 입력 폼(이름, 닉네임, 아이디, 비밀번호, 전화번호, 생년월일, 신체정보, 주소, 프로필 사진)을 출력한다.
3. 고객은 필수 항목을 입력한다. (A1)
4. 시스템은 아이디 중복 여부를 확인한다. (A2)
5. 고객은 서비스 이용약관 및 개인정보 처리방침을 확인하고 동의 항목을 선택한다. (A3)
6. 시스템은 필수 약관 동의 여부를 확인한다.
7. 고객은 [가입 완료] 버튼을 누른다.
8. 시스템은 신규 계정을 생성하고 "회원가입이 완료되었습니다" 메시지를 출력하며 유스케이스를 종료한다.

---

Alternative Path

A1. 필수 항목 미입력 또는 형식 오류
  1. 시스템은 미입력 항목 또는 형식 오류 항목을 강조 표시하고 안내 메시지를 출력한다.
  2. 고객은 해당 항목을 수정하여 재입력한다.
  3. 3단계로 돌아간다.

A2. 아이디 중복 확인 시 이미 사용 중인 아이디
  1. 시스템은 "이미 사용 중인 아이디입니다" 메시지를 출력한다.
  2. 고객은 다른 아이디를 입력한다.
  3. 3단계로 돌아간다.

A3. 필수 약관 미동의
  1. 시스템은 "필수 약관에 동의하셔야 가입이 가능합니다" 메시지를 출력한다.
  2. 고객은 필수 약관 동의 항목을 다시 확인하고 동의를 선택한다.
  3. 8단계로 돌아간다.

---

Exception

E1. 서버 오류로 계정 생성 실패
  1. 시스템은 "일시적인 오류가 발생하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
