# UC04 PurchaseProgram - 헬스 프로그램을 구매한다

## Basic Path

1. 고객은 구매 카테고리 목록에서 [헬스 프로그램] 항목을 선택한다.
2. 시스템은 이용 가능한 헬스 프로그램 목록(요가, 줌바, 다이어트 댄스, 스피닝)과 각 프로그램의 기간, 가격, 담당 트레이너 정보를 출력한다.
3. 고객은 원하는 헬스 프로그램을 선택한다.
4. 시스템은 선택한 프로그램의 상세 정보(운동 정보, 운동 시간, 담당 선생님, 가격)를 출력한다.
5. 고객은 상세 정보를 확인하고 [구매하기] 버튼을 누른다.
6. 시스템은 해당 프로그램의 잔여 정원을 확인한다. (A1)
7. 고객은 프로그램 시작 희망일을 선택한다.
8. 시스템은 선택한 프로그램 정보와 최종 결제 금액을 포함한 주문 확인 화면을 출력한다.
9. 고객은 주문 내역을 확인하고 결제 진행에 동의한다.
10. 시스템은 결제 유스케이스(UC07 Pay)를 실행하고, 결제 완료 후 프로그램을 계정에 등록하며 "헬스 프로그램 구매가 완료되었습니다" 메시지를 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 잔여 정원이 없는 경우
  1. 시스템은 "해당 프로그램은 현재 정원이 마감되었습니다" 메시지와 함께 대기자 등록 안내를 출력한다.
  2. 고객은 다른 프로그램을 선택한다.
  3. 유스케이스를 종료하고 3단계로 돌아간다.

## Exception

E1. 프로그램 정보 로딩 실패
  1. 시스템은 "프로그램 정보를 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 결제 완료 후 프로그램 등록 실패
  1. 시스템은 "프로그램 등록에 실패하였습니다. 고객센터에 문의해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
