# UC14 ManageMyInfo - 내정보를 관리한다

## Basic Path

1. 고객은 앱 메인 화면에서 [내정보] 메뉴를 선택한다.
2. 시스템은 내정보 메인 화면(프로필 정보, 회원권 현황 요약, 포인트 잔액 요약)을 출력한다.
3. 시스템은 회원권 관리 유스케이스(UC16 ManageMembership)를 실행하여 회원권 현황 요약 정보를 출력한다.
4. 시스템은 포인트 조회 유스케이스(UC19 ViewPointHistory)를 실행하여 포인트 잔액 요약 정보를 출력한다.
5. 고객은 내정보 화면에서 표시된 전체 정보를 확인한다. (A1)
6. 고객은 확인하고자 하는 세부 메뉴(회원권 관리, 포인트 조회, 내정보 수정)를 선택한다. (A2)
7. 시스템은 선택한 메뉴에 해당하는 화면 또는 유스케이스로 이동한다.
8. 고객은 해당 메뉴에서 필요한 작업을 완료한다.
9. 고객은 [내정보 메인으로] 버튼을 눌러 내정보 메인 화면으로 돌아온다.
10. 시스템은 업데이트된 내정보 메인 화면을 다시 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 내정보 수정 선택
  1. 고객은 프로필 화면에서 [내정보 수정] 버튼을 선택한다.
  2. 시스템은 내정보 수정 유스케이스(UC15 UpdateMyInfo)를 실행한다.
  3. 수정 완료 후 5단계로 돌아간다.

A2. 회원권 관리 상세 메뉴 선택
  1. 고객은 [회원권 관리] 메뉴를 선택한다.
  2. 시스템은 회원권 관리 유스케이스(UC16 ManageMembership)의 상세 화면으로 이동한다.
  3. 8단계로 돌아간다.

## Exception

E1. 내정보 화면 로딩 실패
  1. 시스템은 "내정보를 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 세션 만료로 인한 접근 불가
  1. 시스템은 "로그인 세션이 만료되었습니다. 다시 로그인해주세요" 메시지를 출력한다.
  2. 유스케이스를 종료한다.
