# UC05 PurchaseEquipment - 운동용품을 구매한다

## Basic Path

1. 고객은 구매 카테고리 목록에서 [운동용품] 항목을 선택한다.
2. 시스템은 판매 중인 운동용품 목록(카테고리, 상품명, 가격, 상세 설명)을 출력한다.
3. 고객은 원하는 운동용품을 선택한다. (A1)
4. 시스템은 선택한 상품의 상세 정보(상품 설명, 가격)를 출력한다.
5. 고객은 구매 수량을 입력한다.
6. 고객은 배송지 정보를 입력한다. (A2)
7. 시스템은 선택 상품, 수량, 수령 방법, 최종 결제 금액을 포함한 주문 확인 화면을 출력한다.
8. 고객은 주문 내역을 확인하고 결제 진행에 동의한다.
9. 시스템은 결제 유스케이스(UC07 Pay)를 실행하고, 결제 완료 후 주문을 등록하며 "운동용품 구매가 완료되었습니다" 메시지와 주문 번호를 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 검색 또는 필터 사용
  1. 고객은 검색어 또는 카테고리 필터(헬스용품, 의류, 보충제)를 입력한다.
  2. 시스템은 필터 조건에 맞는 상품 목록을 출력한다.
  3. 3단계로 돌아간다.

A2. 배송 선택
  1. 시스템은 배송지 입력 폼을 출력한다.
  2. 고객은 배송 받을 주소를 입력하거나 기존 배송지를 선택한다.
  3. 6단계로 돌아간다.

## Exception

E1. 결제 완료 후 주문 등록 실패
  1. 시스템은 "주문 등록에 실패하였습니다. 고객센터에 문의해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 배송지 정보 오류
  1. 시스템은 "올바른 배송지 정보를 입력해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
