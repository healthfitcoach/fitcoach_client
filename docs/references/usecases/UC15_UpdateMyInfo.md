# UC15 UpdateMyInfo - 내정보를 수정한다

## Basic Path

1. 고객은 내정보 화면에서 [내정보 수정] 버튼을 선택한다.
2. 시스템은 수정 가능한 항목(닉네임, 전화번호, 비밀번호, 프로필 사진, 생년월일, 신체정보, 주소)이 포함된 내정보 수정 폼을 출력한다.
3. 고객은 수정하고자 하는 항목을 선택하여 새로운 값을 입력한다. (A1)
4. 시스템은 변경된 항목의 유효성(형식, 중복 여부)을 실시간으로 검증한다. (A2)
5. 고객은 비밀번호 변경을 선택한 경우 현재 비밀번호를 입력하여 본인 확인을 진행한다. (A3)
6. 시스템은 현재 비밀번호의 일치 여부를 확인한다.
7. 고객은 수정 내용을 최종 확인하고 [저장] 버튼을 누른다.
8. 시스템은 수정된 정보를 저장한다.
9. 시스템은 "내정보가 성공적으로 수정되었습니다" 메시지를 출력하며 유스케이스를 종료한다.

## Alternative Path

A1. 프로필 사진 변경
  1. 고객은 [프로필 사진 변경] 버튼을 선택한다.
  2. 시스템은 사진 선택 옵션(갤러리에서 선택, 카메라 촬영)을 출력한다.
  3. 고객은 새로운 프로필 사진을 선택하거나 촬영한다.
  4. 시스템은 선택된 이미지를 미리보기로 출력한다.
  5. 3단계로 돌아간다.

A2. 전화번호 형식 오류 또는 중복
  1. 시스템은 "올바른 형식의 전화번호를 입력해주세요" 또는 "이미 사용 중인 전화번호입니다" 메시지를 출력한다.
  2. 고객은 전화번호를 수정하여 재입력한다.
  3. 4단계로 돌아간다.

A3. 비밀번호 변경 미선택
  1. 시스템은 비밀번호 확인 절차를 생략한다.
  2. 7단계로 돌아간다.

## Exception

E1. 현재 비밀번호 불일치
  1. 시스템은 "현재 비밀번호가 올바르지 않습니다" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 정보 저장 실패
  1. 시스템은 "정보 저장에 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E3. 이미지 업로드 실패
  1. 시스템은 "프로필 사진 업로드에 실패하였습니다. 파일 크기 또는 형식을 확인해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
