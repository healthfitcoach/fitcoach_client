# UC11 SearchEquipment - 기구를 검색한다

## Basic Path

1. 고객은 앱 메인 화면에서 [기구 검색] 메뉴를 선택한다.
2. 시스템은 기구 전체 리스트를 출력한다.
3. 고객은 기구를 선택한다. (A1)
4. 시스템은 선택한 기구의 상세 정보(기구명, 이미지, 간단한 설명)를 출력한다.
5. 고객은 기구 상세 정보를 확인한다. (A2)
6. 시스템은 해당 기구와 관련된 운동 방법 목록을 함께 출력하고 유스케이스를 종료한다.

## Alternative Path

A1. 기구 검색
  1. 고객은 검색창에 기구 이름 또는 키워드를 입력하거나 카테고리(유산소, 근력, 스트레칭)를 선택한다.
  2. 시스템은 입력한 키워드 또는 카테고리에 해당하는 기구 목록을 출력한다.
  3. 3단계로 돌아간다.

A2. 운동방법 조회 선택
  1. 고객은 기구 상세 화면에서 [운동방법 보기] 버튼을 선택한다.
  2. 시스템은 운동방법 조회 유스케이스(UC12 ViewExerciseMethod)를 실행한다.
  3. 5단계로 돌아간다.

## Exception

E1. 기구 상세 정보 로딩 실패
  1. 시스템은 "기구 상세 정보를 불러오는 데 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.
