# UC17 CheckRemainingPeriod - 잔여기간을 조회한다

## Basic Path

1. 시스템은 회원권 관리 유스케이스(UC16)로부터 잔여기간 조회 요청을 수신한다.
2. 시스템은 고객 계정에 등록된 회원권 정보를 조회한다.
3. 시스템은 회원권 시작일과 종료일을 기준으로 잔여 일수를 계산한다. (A1)
4. 시스템은 잔여기간 정보(시작일, 종료일, 잔여 일수, 일시정지 여부)를 화면에 출력한다.
5. 고객은 잔여기간 정보를 확인한다.
6. 시스템은 잔여 일수가 7일 이하인 경우 만료 임박 경고 메시지를 출력한다. (A2)
7. 고객은 회원권 상세 정보(회원권 종류, 이용 가능 시설, 구매일)를 확인하기 위해 [상세 보기] 버튼을 누른다.
8. 시스템은 회원권 상세 정보를 출력한다.
9. 고객은 상세 정보를 확인한다.
10. 시스템은 조회된 잔여기간 정보를 호출한 회원권 관리 유스케이스로 반환하며 유스케이스를 종료한다.

## Alternative Path

A1. 회원권 일시정지 상태
  1. 시스템은 일시정지 기간을 잔여 기간 계산에서 제외하고 실제 남은 이용 가능 일수를 계산한다.
  2. 시스템은 "현재 회원권이 일시정지 상태입니다" 메시지와 함께 일시정지 기간 및 재개 예정일을 출력한다.
  3. 4단계로 돌아간다.

A2. 잔여 기간이 0일 또는 만료 상태
  1. 시스템은 "회원권이 만료되었습니다" 메시지와 함께 [회원권 구매하기] 버튼을 출력한다.
  2. 고객은 [회원권 구매하기] 버튼을 눌러 회원권 구매 유스케이스(UC03)로 이동하거나 화면을 닫는다.
  3. 유스케이스를 종료한다.

## Exception

E1. 회원권 데이터 조회 실패
  1. 시스템은 "회원권 잔여기간 조회에 실패하였습니다. 잠시 후 다시 시도해주세요" 오류 메시지를 출력한다.
  2. 유스케이스를 종료한다.

E2. 날짜 계산 오류
  1. 시스템은 오류 로그를 기록하고 관리자에게 알림을 발송한다.
  2. 시스템은 "일시적인 오류가 발생하였습니다. 고객센터에 문의해주세요" 메시지를 출력한다.
  3. 유스케이스를 종료한다.
